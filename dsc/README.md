# Sony DSC 仕入れ判定ツール

メルカリの売り切れ検索をもとに、Sony DSCシリーズの型番、カラー、仕入れ金額から利益額、利益率、仕入れ判定を計算する静的Webアプリです。

## 使い方

`index.html` をブラウザで開くだけで使えます。型番、カラー、相場モードを選ぶと販売見込みが自動入力され、仕入れ金額を入れると判定が更新されます。

## 相場条件

- 対象期間: 2026-02-27 から 2026-05-27
- 対象: メルカリ売り切れ商品、デジタルカメラカテゴリ相当、Sony DSC型番
- 価格: 各型番の中央値
- 状態: 未使用に近い、目立った傷や汚れなしを「目立った傷なし」側に集約
- ジャンク: タイトルにジャンク、訳あり、難あり、故障、動作未確認などを含むものを集計
- 判定条件、送料、手数料、カラー補正、UIは昨日版のCanon IXYアプリと同じ構成

## ファイル

- `index.html`: 画面
- `styles.css`: スタイル
- `app.js`: 相場データと計算ロジック
